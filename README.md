# 时光相框 · TimeFrame

一款沉浸式拍立得风格的在线相机应用，让拍照更有仪式感。

## 项目简介

时光相框是一款模拟真实拍立得体验的 Web 应用，通过「摇一摇显影」的独特交互，让用户在数字化时代也能感受到传统相机的仪式感和惊喜感。

## 核心功能

### 🎯 摇一摇显影
- 模拟拍立得显影过程
- 桌面端：鼠标晃动显影
- 移动端：设备摇晃显影
- 实时进度指示

### 🎨 主题拍照模式
- 派对模式：活力四射的氛围
- 复古模式：怀旧风格
- 自然模式：清新自然
- 极简模式：黑白风格
- 梦幻模式：柔和色调

### 📷 丰富的滤镜和相框
- 9种滤镜效果（原片、黑白、赤红、日落、复古、森林、海蓝、紫罗兰、彩虹）
- 7种相框样式（经典白边、复古黄边、现代黑边、马卡龙粉、海洋蓝、派对红、森林绿）
- 多种字体选择

### 🎮 其他功能
- 文字添加：为照片添加心情描述
- 本地存储：照片保存在本地浏览器
- 照片删除和下载
- 深色/浅色主题切换

## 技术栈

- HTML5：页面结构
- CSS3：样式和动画
- 原生 JavaScript：核心逻辑
- WebRTC：摄像头访问
- Canvas API：图像处理和合成
- DeviceMotion API：设备运动检测（移动端）
- localStorage：本地存储

## 快速开始

### 本地运行

1. 克隆仓库
```bash
git clone https://github.com/may-ai-007/classictimeframe.git
cd classictimeframe
```

2. 使用本地服务器运行（推荐）

使用 Python：
```bash
python3 -m http.server 8000
```

或使用 Node.js（需要安装 http-server）：
```bash
npx http-server -p 8000
```

3. 打开浏览器访问
```
http://localhost:8000
```

### 使用说明

1. 打开应用，允许访问摄像头权限
2. 在输入框中写下心情文字（可选）
3. 选择滤镜和相框（或使用主题模式）
4. 点击快门拍照
5. 摇晃设备或晃动鼠标，等待照片显影
6. 双击照片保存，或点击 × 删除
